Luogu P4544 [USACO10NOV]购买饲料Buying Feed

发表于 2019-11-14 分类于 题解

单调队列优化dp

把终点看成一个商店，只不过$ X_i=E$ , $F_i=C_i=0$

把所有商店按$X$排序。

用 $f[i][j]$ 表示到了第 $i$ 个商店，买了 $j$ 个物品的最小花费,枚举到 第$i$个商店之前的物品个数 $k$ ，$D_i$ 表示第 $i$ 个商店与第 $i-1$ 个商店的距离，则有:

$f[i][j]=\min_{k\le j} f[i-1][k]+D_{i+1}\times j^2 + (j-k)\times C_{i}$

把与 $k$ 有关的项放在一起：

$f[i][j]=\min_{k\le j} (f[i-1][k]-k\times C_{i})+D_{i+1}\times j^2 + j\times C_{i}$

把$f[i-1][k]-k\times C_{i}$用单调队列维护

反正$j$扔进去以后也会被更大的$j$继承

牛客CSP-S提高组1T2 乃爱与城市拥挤程度

发表于 2019-11-14 分类于 题解

树形dp(up&down)

先考虑第一个问，设$c[u][k]$为$u$子树内(包括$u$)与$u$距离不超过$k$的节点数，初始化为$1$,则有:

$c[u][k]= \sum c[v][k-1] $

再考虑子树外对答案的贡献，暴力往上跳k层，设当前跳到的节点为 $p$ ,则有:

$ans=c[u][k]+\sum_{i=1}^{k} {c[fa[p]][k-i]-c[p][k-i-1]}$

考虑第二个问,设$f[u][k]$为$u$子树内(包括$u$)拥挤度的乘积，则有:

$f[u][k]=c[u][k] * \prod f[v][k-1]$

设$d[u][k]$为$u$以外(不包括$u$)与$u$距离不超过$k$的节点数, $d[u][0]=0$ , $d[u][1]=1$ ,则有:

$d[u][k]=d[fa[u]][k-1]+c[fa[u]][k-1]-c[u][k-2]$

暴力往上跳k层，设当前跳到的节点为 $p$ ,考虑 $fa[p]$对答案的贡献，则有:

$ ans=f[u][k](c[u][k]+d[u][k]) \prod_{i=1}^{k}\frac{f[fa[p]][k-i](d[fa[p]][K-i]+c[fa[p]][K-i]-c[p][K-i-1])}{f[p][k-2]c[fa[p]][K-i]}$

$(d[fa[p]][K-i]+c[fa[p]][K-i]-c[p][K-i-1])$即为$fa[p]$的拥挤度

$\frac{f[fa[p]][k-1]}{f[p][k-2]*c[fa[p]][k-i]}$即为$fa[p]$子树内除了当前子树和 $fa[p]$的拥挤度的乘积

红色部分拥挤度的乘积即为$fa[p]$对答案的贡献

把分子乘在一起，把分母乘在一起，就只需做一次逆元

dp

设$f[i]$为到第$i$个点的最小答案，则有:

$f[i]=min(f[i],f[j]+1)$

转移条件为 $j$ 与 $i$ 的连线的斜率要小于等于 $j$ 到 $i$ 之间所有点与 $i$ 的连线的斜率

$j$ 的高度甚至可以比 $i$ 的高度高

你甚至可以移项变形成乘法

Luogu P2344 奶牛抗议

发表于 2019-11-14 分类于 题解

令 $f[i]$ 表示到 $i$ 的方案数

有$f[i] = \sum f[j]$ ($\sum_{k = j+1}^i a[k] \geq 0$)

写成前缀和的形式就是

$f[i] = \sum f[j]$ ($sum[i]-sum[j]\geq 0$)

拿$sum$当下标，$ \leq sum[i]$ 的 $f[j]$ 的和可以使用树状数组维护

$f[0] = 1$,优先插入

Luogu U89718 序列扩展

发表于 2019-11-14 分类于 题解

数学

考虑数组为 {2, 2, 2} 的情况，则一次操作后依然是 {2, 2, 2}

不难发现，当某个因子出现在至少三个数中时，这一因子就再也无法消去,所以操作数最多为两次且为因子最多出现的次数，否则都是正无穷

只需要考虑质因子的贡献即可,先欧拉筛，再$O(\frac{\sqrt N}{\log N})$ 找出一个数所有的质因子,或者考虑每个质数，看它是多少个数的因子

数据有锅！！！
